Зависимос
### Зависимости для сборки

> - Git
> - .NET SDK 9.0.101

### Windows

> 1. Клонируйте этот репозиторий
> 2. Запустите в терминале `git submodule update --init --recursive`, чтобы загрузить движок
> 3. После внесения любых изменений в исходный код запустите `Scripts\bat\buildAllRelease.bat`
> 4. Запустите `Scripts\bat\runQuickAll.bat`, чтобы запустить клиент и сервер
> 5. Подключитесь к localhost в клиенте и играйте

### Linux / MAC

> 1. Клонируйте этот репозиторий
> 2. Запустите в терминале `git submodule update --init --recursive`, чтобы загрузить движок
> 3. После внесения любых изменений в исходный код запустите `Scripts/sh/buildAllRelease.sh`
> 4. Запустите `Scripts/sh/runQuickAll.sh`, чтобы запустить клиент и сервер
> 5. Подключитесь к localhost в клиенте и играйте
